# Auto-SignIn 设计规范文档

## 1. 设计风格

- **美学风格**: 现代简约 + 卡片式设计
- **核心理念**: 清晰、直观、易用
- **关键特征**:
  - 清晰的视觉层级
  - 友好的表单引导
  - 即时的操作反馈
  - 响应式布局

## 2. 色彩方案

### 主色调
- **主色**: `#667eea` (紫蓝渐变起始)
- **次色**: `#764ba2` (紫蓝渐变结束)
- **背景**: `#f5f7fa` (浅灰背景)
- **卡片背景**: `#ffffff` (纯白)

### 功能色
- **成功**: `#10b981` (绿色)
- **警告**: `#f59e0b` (橙色)
- **错误**: `#ef4444` (红色)
- **信息**: `#3b82f6` (蓝色)

### 文字色
- **主文字**: `#1f2937` (深灰)
- **次文字**: `#6b7280` (中灰)
- **提示文字**: `#9ca3af` (浅灰)
- **反白文字**: `#ffffff`

## 3. 字体规范

### 字体族
- **主字体**: `-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`
- **代码字体**: `"SF Mono", Monaco, "Cascadia Code", Consolas, monospace`

### 字号层级
- **H1**: 32px / 2rem (页面标题)
- **H2**: 24px / 1.5rem (卡片标题)
- **H3**: 18px / 1.125rem (小节标题)
- **Body**: 16px / 1rem (正文)
- **Small**: 14px / 0.875rem (辅助文字)
- **Tiny**: 12px / 0.75rem (标签、提示)

### 字重
- **Regular**: 400 (正文)
- **Medium**: 500 (强调)
- **Semibold**: 600 (标题)
- **Bold**: 700 (重要标题)

## 4. 图标策略

- **图标库**: Heroicons (Outline风格)
- **描边宽度**: 1.5px
- **尺寸**: 16px / 20px / 24px
- **颜色**: 继承父元素或使用功能色

## 5. 间距系统

- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px
- **2xl**: 48px

## 6. 圆角规范

- **按钮**: 8px
- **输入框**: 8px
- **卡片**: 16px
- **标签**: 20px (胶囊形)
- **模态框**: 16px

## 7. 阴影层级

- **sm**: `0 1px 2px 0 rgba(0, 0, 0, 0.05)`
- **md**: `0 4px 6px -1px rgba(0, 0, 0, 0.1)`
- **lg**: `0 10px 15px -3px rgba(0, 0, 0, 0.1)`
- **xl**: `0 20px 25px -5px rgba(0, 0, 0, 0.1)`

## 8. 动画规范

- **快速**: 150ms (hover效果)
- **正常**: 300ms (展开/收起)
- **慢速**: 500ms (页面过渡)
- **缓动函数**: `cubic-bezier(0.4, 0, 0.2, 1)`

## 9. 表单设计规范

### 输入框
- 高度: 40px
- 内边距: 12px 16px
- 边框: 1px solid #e5e7eb
- 聚焦边框: 2px solid #667eea
- 错误边框: 2px solid #ef4444
- 占位符颜色: #9ca3af

### 标签
- 字号: 14px
- 字重: 500
- 颜色: #374151
- 底部间距: 4px

### 提示文字
- 字号: 12px
- 颜色: #6b7280
- 顶部间距: 4px

### 错误提示
- 字号: 12px
- 颜色: #ef4444
- 图标: 警告图标

## 10. 按钮规范

### 主要按钮
- 背景: 渐变 `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- 文字: 白色
- 高度: 40px
- 内边距: 12px 24px
- 悬停: transform: translateY(-1px), box-shadow增强

### 次要按钮
- 背景: 白色
- 边框: 2px solid #667eea
- 文字: #667eea
- 悬停: 背景 #f3f4f6

### 禁用状态
- 背景: #e5e7eb
- 文字: #9ca3af
- 光标: not-allowed

## 11. 响应式断点

- **Mobile**: < 640px
- **Tablet**: 640px - 1024px
- **Desktop**: > 1024px

## 12. 无障碍设计

- 所有交互元素支持键盘操作
- 表单字段关联label
- 错误提示使用role="alert"
- 颜色对比度符合WCAG AA标准
- 加载状态提供屏幕阅读器提示
